# Empieza aquí

1. Copia `.env.example` a `.env`.
2. Cambia tu correo, contraseña y secretos.
3. Ejecuta:

```bash
npm install
npm run seed
npm start
```

4. Entra a `http://localhost:3000`.
5. Agrega tus productos favoritos.
6. Para subir a GitHub:

```bash
git init
git add .
git commit -m "DealWatch MX inicial"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/dealwatch-mx.git
git push -u origin main
```
