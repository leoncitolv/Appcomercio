Importante

Por qué esta versión sí sirve en GitHub Pages

Esta app está hecha como HTML estático: index.html + CSS + JS + manifest. Súbela a GitHub Pages y se abre como app instalable.

Limitación honesta: Amazon, Mercado Libre y PlayStation normalmente no permiten leer precios directo desde un HTML por bloqueos de seguridad del navegador. Esta versión guarda tus productos y calcula ofertas cuando actualizas el precio. Para monitoreo automático real se agrega después un backend o GitHub Actions.
  1. Sube todos estos archivos a la raíz del repo.
  2. En GitHub entra a Settings → Pages.
  3. Selecciona Deploy from a branch, rama main, carpeta /root.
  4. Abre tu URL usuario.github.io/repositorio.
  5. En celular usa Compartir → Agregar a pantalla de inicio.
